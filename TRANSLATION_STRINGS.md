## NOTES FOR TRANSLATORS

1. **Technical Terms**: Physics and chemistry terms like "pH", "Coulomb", "Gibbs", "Lorentz" should be transliterated appropriately for Arabic
2. **Units**: Scientific units (m/s, kg, J, etc.) remain the same in both languages
3. **Superscripts/Subscripts**: Chemical notation like [H⁺], [OH⁻], ΔG, etc. should be preserved as-is
4. **Modal Quotes**: Some text appears in dialog boxes and needs consideration for RTL layout if translating to Arabic
5. **Context**: Many strings have short labels paired with longer descriptions - both need translation
6. **Dynamic Content**: Some strings include variables (e.g., "{history.length} reactions") - preserve the placeholder syntax
